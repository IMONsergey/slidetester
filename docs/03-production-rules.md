# Production Rules

- Do not modify existing Figma frames.
- Do not modify Icons frame.
- Generate new frames in a separate column.
- Keep generated frames editable.
- Top-level composition uses absolute coordinates.
- Internal Auto Layout is allowed for cards, pills and rows.
- Every slide needs a source reference.
- Every slide needs a visual fidelity score.
- `DESIGN_FIDELITY_ADDENDUM.md` overrides any generic design shortcut.
