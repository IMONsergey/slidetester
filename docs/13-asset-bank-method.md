# Asset Bank Method

The asset bank indexes source-deck visuals that are safe to reuse without inheriting irrelevant business content.

## Safe reuse classes

- backgrounds
- glows
- chart shells
- card shells
- line systems
- logo / footer systems
- icon sources

## Unsafe reuse classes

- old chart traces as data
- old legends
- old footnotes
- old years
- old competitor names
- topic-specific old statements

## Method

1. Capture or reuse audited source nodes.
2. Classify each asset by semantic risk.
3. Allow direct reuse only for neutral visuals.
4. Force rebuild for any semantic content.
